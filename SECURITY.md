# Security Policy

If you discover a security vulnerability in this package, please report it privately.

## Reporting

- Open a private security advisory on GitHub for this repository, or
- Contact the maintainers directly via the channels listed on the project profile.

Please include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact and affected versions

We will acknowledge reports as quickly as possible and coordinate a responsible disclosure.
